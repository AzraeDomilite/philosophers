Handles eating and sleeping cycle for a philosopher.

Parameters
philoPhilosopher structure

Sequence:

  1. Acquires both forks
  2. Eats for specified duration
  3. Updates meal count and timestamp
  4. Sleeps for specified duration
  5. Releases forks
